# 서브공간을 그리드 셀로 분리 (간소화 버전)

각 VenueCard 내부의 서브공간(A/B/C/D)을 평면도처럼 **그리드 셀**로 분할하고, 셀 안에는 **세션명 + 주문/수령 숫자**만 표시.

## 변경 파일
- `src/routes/admin.tsx` — `VenueCard`의 서브공간 렌더링 부분만

## 그리드 분할 규칙
- 2칸(402) → `1×2`
- 3칸(403, 404, hall) → `1×3`
- 4칸(401) → `2×2`

```text
┌────┬────┐    ┌──┬──┬──┐    ┌────┬────┐
│ A  │ B  │    │A │B │C │    │ A  │ B  │
└────┴────┘    └──┴──┴──┘    ├────┼────┤
   402         403/404/hall   │ C  │ D  │
                              └────┴────┘
                                  401
```

## 각 셀 내용 (간결)
- 상단: 코드 `A`/`B`/... + 작은 라벨(예: `401-A`)
- 중단: 현재 진행 세션명 (2줄 클램프, 없으면 `—`)
- 하단: `주문 N` · `수령 N`

진행률 막대·수용 인원·수령률 막대는 셀에서 제거(상단 합계 영역에만 유지).
