## 문제
세 트럭(민트·망고·블루베리)의 우측 뒤쪽 유리창 2개가 청중(딸기) 트럭과 형태·크기·간격·프레임이 다릅니다. 특히 맨 뒤(우측) 창이 가장 차이가 큽니다.

## 작업
1. 청중 트럭(`truck-strawberry`)에서 우측 2분할 창 영역(좌측 넓은 창 + 우측 좁은 창, 프레임 포함)을 픽셀 단위로 크롭 → 마스크 추출
2. 추출한 우측 창 영역을, 민트·망고·블루베리 트럭 각각의 정확히 동일한 좌표에 합성
   - 합성 전 기존 우측 창 영역을 차체색으로 덮어 잔상 제거
   - 창 내부 색은 청중과 동일한 아이시 화이트/페일블루 유지(다른 두 창과 통일성 위해)
   - 프레임 색만 각 트럭 차체 톤(민트/오렌지/라벤더)에 살짝 맞춤
3. 도어창·중앙창은 그대로 유지(이미 청중과 동일)
4. 결과 확대 검증 — 세 트럭의 우측 창이 청중과 완전히 동일한 모양/위치/간격인지 확인
5. `truck-mint.png`, `truck-mango.png`, `truck-blueberry.png` 자산만 교체

## 방식
픽셀 합성(PIL) 우선. 청중 트럭의 우측 창 RGBA 패치를 그대로 복사-붙여넣기 방식. AI 재생성은 사용하지 않음(형태 변형 방지).
