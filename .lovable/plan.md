## 셰이딩 레이어 재구성 (z-stack 아래→위)

마스크는 정사각 영역. 본체(반구)는 대략 위 0%~68%, 스커트는 65%~100%로 본다.

1. **베이스 그라데이션** (기존 유지)

2. **반구 볼륨 셰이딩 (강화)** — 본체 영역만 덮는 큰 radial로 위는 환하게, 본체 아래쪽은 진하게
   ```
   background: radial-gradient(ellipse 75% 65% at 35% 25%,
     rgba(255,255,255,0.55) 0%,
     rgba(255,255,255,0.15) 28%,
     rgba(255,255,255,0) 45%,
     rgba(0,0,0,0.18) 65%,
     rgba(0,0,0,0.28) 78%);
   /* 본체 영역만 적용 */
   clip-path: inset(0 0 35% 0);
   ```

3. **반구 specular 하이라이트** — 작고 선명한 빛 반사점 (28% 22% 위치)
   `radial-gradient(circle 70px at 28% 22%, rgba(255,255,255,0.95), transparent 70%)` + `clip-path: inset(0 0 40% 0)`

4. **스커트 분리 경계선 (신규, 핵심)** — 본체 바닥의 그림자 띠
   본체와 스커트 사이 약 62~70% 영역에 진한 어두운 띠 + 바로 아래 살짝 밝은 림으로 **물리적 단차**가 보이도록.
   ```
   background:
     linear-gradient(to bottom,
       transparent 60%,
       rgba(0,0,0,0.0) 60%,
       rgba(0,0,0,0.35) 66%,
       rgba(0,0,0,0.22) 70%,
       rgba(255,255,255,0.15) 72%,
       transparent 78%);
   ```

5. **스커트 음영 (찌글찌글한 부분)** — 스커트 영역만 살짝 어둡고 채도↓
   ```
   background: linear-gradient(to bottom,
     transparent 65%,
     rgba(0,0,0,0.10) 75%,
     rgba(0,0,0,0.20) 100%);
   ```

6. **엣지 비네팅 (유지)** — 가장자리 림 음영

7. **토핑 + 컨텐츠** (유지) — 컨텐츠 패딩은 본체 영역에 맞도록 `paddingBottom: 32%`로 조정해 텍스트가 본체 안에 머물게.

## 기대 효과
- 상단 큰 하이라이트 + 본체 아래쪽 짙어지는 셰이딩 → 반구처럼 볼록한 입체감
- 본체 바닥의 짙은 띠 + 얇은 밝은 선 → 본체와 흘러내린 스커트가 분리된 듯한 단차
- 스커트는 약간 어둡고 가라앉아 보여 본체가 더 도드라짐
