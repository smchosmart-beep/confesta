## 요약
`IceCreamCone.tsx`의 와플콘을 3D 입체 카라멜 톤으로 다시 디자인한다. 스쿱·토핑·빈 상태(고스트) 로직은 그대로.

## 변경 대상
- `src/components/confesta/IceCreamCone.tsx` — 와플콘 영역(약 67~109행)만 교체

## 디자인 방향
**3D 입체 와플콘 · 카라멜 톤** (`#F4D9A8 → #D9A86A → #8E5A24 → #4A2A0E`)

현재 문제점: 단색 그라데이션 위에 45/-45도 반복 라인으로 격자 패턴 → 평면적이고 "프린트한 종이" 느낌.

새 구성:
1. **콘 실루엣**: 기존 삼각형 `clipPath` 유지 (레이아웃 안정).
2. **베이스 색**: 카라멜 그라데이션 (밝은 상단 → 깊은 다크 카라멜 하단), 측면에 살짝 어두운 라디얼 음영을 더해 원기둥성(立体感) 표현.
3. **와플 격자 (SVG 패턴)**:
   - 기울어진 다이아몬드 셀(rhombus) 격자를 SVG `<pattern>`으로 생성.
   - 각 셀에 inner shadow(어두운 안쪽 + 밝은 가장자리)를 그려 셀이 "움푹 파인" 듯한 입체감.
   - 콘의 폭이 아래로 좁아지므로 셀도 살짝 원근감 있게 작아지도록 종횡비 변화(또는 perspective skew).
4. **하이라이트 & 음영**:
   - 좌측 30%에 부드러운 흰색 하이라이트 (광택).
   - 우측 30%에 다크 카라멜 음영 (자연광).
   - 콘 끝(아래 꼭짓점)을 더 진한 색으로 강조해 구운 느낌.
5. **상단 림(rim)**: 두 겹 — 외곽은 밝은 카라멜 띠, 내부는 살짝 어두운 그림자로 두께감.
6. **드롭 섀도우**: 기존 `drop-shadow(0 6px 10px rgba(110,68,22,0.25))` 유지하되 컬러 약간 진하게.

## 기술 노트
- SVG `<pattern patternUnits="userSpaceOnUse">`로 다이아몬드 격자를 정의하고, 콘 영역(clip-path와 동일한 path)에 채워 넣음.
- 격자 내부 음각 효과는 각 다이아몬드에 `<linearGradient>` (밝은 좌상 → 어두운 우하) + 1px 짙은 외곽선으로 구현. CSS `filter` 사용 안 함(성능).
- 카라멜 톤은 디자인 토큰화하지 않고 컴포넌트 내부 상수로 유지 (다른 곳에서 재사용 안 함).
- 사이즈는 `size` prop에 비례(현재처럼 `coneW = size * 0.78`).

## 기대 결과
손으로 구운 듯한 입체 와플콘. 단순 반복 라인 대신 음영 있는 다이아몬드 격자로 사실감 + 세련됨을 동시에 확보.
