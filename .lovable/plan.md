# 스쿱 카드 모양 재작업 — "공 + 얇은 받침"

## 현재 문제 (스크린샷 비교)
| 항목 | 현재 카드 | 목표 (첨부 이미지) |
|---|---|---|
| 비율 | 4:3 가로로 넓음 | 거의 1:1 (약간 세로로 김) |
| 본체 | 가로로 늘어진 낮은 돔 | **거의 완전한 원(공 모양)** |
| 아랫부분 | 카드 폭 전체에 큰 물결 드립 | 공 바로 아래, 폭의 ~90%로 좁고 **얇은 녹은 받침**(살짝 비대칭) |

## 새 실루엣
```text
       _.--""""--._
     .'            '.
    /                \
   |    (스쿱 본체)    |     ← 거의 원
    \                /
     '._          _.'
       \_/~^~\_/~^\        ← 폭 좁고, 얇고, 비대칭 작은 물결 받침
```
